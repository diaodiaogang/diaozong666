## 常用指令

### 从工作区—>暂存区			

**git add .**

### 从暂存区—>本地仓库         

**git commit "提交的信息比如修改了什么功能"**

### 查看状态                          

**git status**

### 查看提交的记录                 

**git log     git -log**

### 版本回退(理解即可)           

 **git reset -heard <commitID>**

### 分支操作

#### 查看分支                     

**git branch**

#### 查看并创建分支            

**git checkout -b  分支名**

#### 合并分支                     

**git merge 分支名**
